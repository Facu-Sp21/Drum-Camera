# Drum-Camera
Es un proyecto personal en el cual quiero detectar movimientos con la camara y transformarlo en sonidos de bateria, y me permita tocar la bateria en el aire solo con la camara

los principales desafios de este proyecto son:

- reducir la latencia de los golpes lo maximo posible para que se sienta una experiencia real.
- Tener una correcta lectura de los inputs para no perder golpes.
- implementar funciones que detecten velocidad/fuerza para que halla sensibilidad dependiendo el tipo de golpe
- Aprender herramientas de desarrollo Android.

  MPV:
  lo minimo que quiero lograr es poder tener caja, bombo y hit-hat.


  Tecnologias:
  -python (para prototipado y validacion del proyecto)
  -Kotlin (para desarrollo general del backend)
  -OpenCV (deteccion de backetas y zonas)
